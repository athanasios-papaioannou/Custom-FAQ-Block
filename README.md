# Custom FAQ Block

An interactive Gutenberg FAQ block for WordPress built with React and the WordPress Block Editor APIs.

The block provides a modern FAQ management experience directly inside Gutenberg with accordion functionality on the frontend.

---

## Features

- Add FAQ items
- Edit existing FAQs
- Delete FAQs with confirmation
- Reorder FAQ items
- Expand / Collapse accordion functionality
- Expand All / Collapse All controls
- Rich text support
- Interactive frontend behavior
- Smooth accordion animations
- Customizable button colors
- Responsive UI
- Modern Gutenberg editor experience

---

## Technologies Used

- React
- Gutenberg Block API
- WordPress Block Editor
- JavaScript (ESNext)
- SCSS
- Webpack / @wordpress/scripts

---

## Installation

1. Download the plugin ZIP
2. Go to WordPress Admin → Plugins → Add New
3. Upload the ZIP file
4. Activate the plugin
5. Open the Gutenberg editor
6. Search for: Custom FAQ Block
