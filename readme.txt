An interactive Gutenberg FAQ block with accordion functionality, rich text editing, reorder controls, and frontend interactivity.

== Description ==

Custom FAQ Block is a modern Gutenberg block for creating interactive FAQ sections directly inside the WordPress block editor.

Features include:

- Add / Edit / Delete FAQs
- Reorder FAQ items
- Rich text support
- Expand / Collapse accordion
- Expand All / Collapse All controls
- Smooth frontend animations
- Button color customization
- Responsive design

Built using React and the WordPress Block Editor APIs.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/custom-faq-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress.
3. Open the Gutenberg editor.
4. Search for "Custom FAQ Block".

== Frequently Asked Questions ==

= Does the block support rich text formatting? =

Yes. Questions and answers support rich text editing.

= Can I reorder FAQ items? =

Yes. FAQ items can be reordered using built-in controls.

= Is the frontend interactive? =

Yes. The frontend includes accordion functionality with smooth transitions.
